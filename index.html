Project

DetectiveKite

DetectiveKite is a free perch mount camera trap images auto-detection system which is developed base on yolov5s, a light and fast object detection architecture. I believe DetectiveKite can help biologists in Taiwan to accelerate the heavy species labeling works.

遙測追蹤文獻查詢網站

利用網頁爬蟲技術，蒐集了補份鳥類遙測追蹤的文獻，架設了一個可以依照內文、發報器定位技術、傳輸技術、重量、物種分類來查詢文獻。網頁後端： Python Flask、MySQL。

關注物種點位回報系統

因計畫需求，需要請其他單位的同事以及各地外聘調查員回報關注鳥種的點位及數量，但因為各自將所記錄的點位回報，格式相當不統一，也無法確保回報資料品質，因次我設計了一個以 Telegram bot 為使用者介面的關注物種點位回報系統。使用者只須與 Telegram bot 帳號對話，即可將關注物種點位資料回傳至我自己架設的 MySQL 伺服器。
